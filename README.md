# BT-哔哩哔哩Tv客户端
BT - Bilibili TV Client

**最低支持**：  
Android 4.4

安卓>5.1 universal.apk  
安卓4.4~5.1 universal-api19-22.apk  

**特色**：  
发弹幕，回复评论，音量均衡，AI原声翻译，图文/专栏，互动弹幕，人像防挡，多账号，开发者模式
